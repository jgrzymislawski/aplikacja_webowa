# 🚀 React + TypeScript App

Aplikacja webowa do zarządzania i dzielenia się wydatkami stworzona przy użyciu **React + TypeScript**.
Projekt zawiera podstawowy system logowania oraz routing między stronami.

---

## 📦 Technologie

- React
- TypeScript
- Vite
- React Router

---

## 🔧 Instalacja i uruchomienie

1. Zainstaluj zależności:

```
npm install
```

2. Uruchom aplikację:

```
npm run dev
```

3. Otwórz w przeglądarce:

```
http://localhost:5173/
```
